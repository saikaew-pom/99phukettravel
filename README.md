# 99event.co — Discover the Real Phuket

**99event.co** is a modern, high-performance travel authority site dedicated to uncovering the authentic side of Phuket, Thailand. Built with a focus on local expertise, editorial-quality content, and interactive utility.

## 🚀 Tech Stack

- **Framework:** [Astro 6.0](https://astro.build/) (Static Site Generation)
- **UI Library:** [React 19](https://react.dev/) (Interactive components)
- **Styling:** [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Content:** MDX (Markdown with JSX) for rich, interactive blog posts.

## ✨ Key Features

- **Beach Finder:** An interactive filtering system to find the perfect Phuket beach based on "vibe" (Party, Family, Luxury, etc.) and activities (Snorkeling).
- **Interactive Tools:**
  - **2026 Visa Checker:** Instantly identifies visa exemption rules and mandatory forms (TDAC) for 90+ countries.
  - **Budget Calculator:** Generates personalized daily cost estimates in multiple currencies (USD, EUR, GBP, AUD, THB).
- **Authority Content:** Long-form, SEO-optimized guides for major beaches (Patong, Kata, Bang Tao, etc.) and essential travel tips.
- **SEO Optimized:** 
  - Automated Sitemap generation.
  - Advanced JSON-LD Schema (Article, WebSite, Breadcrumb).
  - Fully responsive and accessible design.

## 📂 Project Structure

```text
/
├── public/              # Static assets (High-quality generated images)
├── src/
│   ├── components/      # React & Astro interactive components
│   ├── content/         # MDX blog posts organized by collection
│   ├── data/            # Static data stores (Beach data, etc.)
│   ├── layouts/         # Base and Post layouts
│   └── pages/           # Dynamic and static routes
├── astro.config.mjs     # Astro & Vite configuration
└── package.json         # Dependencies and scripts
```

## 🧞 Commands

| Command | Action |
| :--- | :--- |
| `npm install` | Installs dependencies |
| `npm run dev` | Starts local dev server at `localhost:4321` |
| `npm run build` | Build the production site to `./dist/` |
| `npm run preview` | Preview the build locally |
